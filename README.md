SA University - Premium package (Style C)

How to run locally:
1. unzip
2. npm install
3. npm start
4. open http://localhost:3000
5. if first time visit /setup to create admin

Notes:
- This package uses sqlite (database.sqlite created automatically).
- Uploaded files go to public/uploads.
- Socket.io provides live chat (front connects to /socket.io).
- Package.json corrected for Render compatibility (multer version fixed).
